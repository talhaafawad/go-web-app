Troubleshooting Highlights

Issue: Encountered DNS resolution errors and connection timeouts when accessing apps or Git repositories via ArgoCD.

Resolution: Investigated and adjusted EKS security groups to allow outbound UDP on port 53, reconfigured network ACLs, and recreated the Kubernetes cluster with improved settings using a /manage configuration option.

Additionally, I reinstalled ArgoCD and ingress controllers, and updated local hosts files for testing with the LoadBalancer's external IP. These steps resolved connectivity issues and stabilized deployments.
